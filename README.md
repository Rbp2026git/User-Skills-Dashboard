# User-Skills-Dashboard
<h2>User Skills Dashboard<</h2>
<h3>Real-world scenario</h3>
<p>
   Suppose you are building a profile page for a developer.
   Skills are stored in an object, but the UI needs them as an array so they can be displayed or processed.
</p>
<p><b>We will:</b></p>
<p>1. Store skills in an object</p>
<p>2. Use for...in to iterate over the object</p>
<p>3. Use Array.from() to convert values into an array</p>
<p>4. Display them</p>
